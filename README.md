## 🎨 Animations Used

| Animation | Effect |
|-----------|--------|
| `fadeIn` | Whole container fades in on load |
| `glow` | Heading glows blue infinitely |
| `slideUp` | Subtitle slides up after 1s delay |
| `float` | Bubbles float from bottom to top |
